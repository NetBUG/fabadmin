# Installing
Assuming you have `python` and `pip` working.
Clone the repo
mkdir -r /w/fabadmin/current
mkdir /w/fabadmin/.logs/
mkdir /w/fabadmin/env
pip install virtualenv
`virtualenv env`
`source env/bin/activate`
`pip install -r requirements.txt`
